## placement new(定位new)

- 一种对`new`的重载，不支持用户自定义

- ```c++
  //调用类的构造函数并将值存在ptr处
  //new (ptr) classname(parameter);
  int *b = new int(1);
  new (b) Base();
  Base *pb = b;
  ```



## 默认构造函数

- C++提供默认构造函数时，如果类中没有对象（高级数据类型），则不会进行默认初始化，这时候可能出现"C4700"使用了未初始化的局部变量



## 左值右值

- 左值：可以取地址、有名字
- 右值
  - 将亡值：将要被移动的对象
  - 纯右值：非返回引用函数返回的临时变量、一些运算表达式、一些字面量
